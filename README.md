**SimUI是一款绿色、轻量级、配置自由的模拟游戏ROM管理软件。**

软件支持自由添加游戏平台、支持多游戏模拟器、多游戏目录； 

支持ROM别名，ROM子游戏； 

支持展示图、场景图显示、支持gif动画；  

支持游戏简介、游戏攻略，支持富文本显示，支持txt、markdown、html三种语法，并可以混合运用；  

支持自定义皮肤、支持多语言；

还支持懒加载、软件改名、一键下载缩略图、启动电子书等诸多功能。

**软件完全免费，你可随意制作合集、并进行传播。**
